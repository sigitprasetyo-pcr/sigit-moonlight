# PENGEMBANGAN APLIKASI NUSADATA - BINA DESA DIGITAL

## PLATFORM

Android Studio

Menggunakan:

* Kotlin
* XML Layout
* Fragment
* RecyclerView
* Room Database
* ViewBinding
* Material Design 3

---

# TUJUAN

Lakukan pengembangan aplikasi NusaData dengan menerapkan Room Database pada 2 menu utama.

Implementasi Room Database harus benar-benar digunakan untuk penyimpanan data lokal.

Jangan menggunakan ArrayList sebagai sumber data utama.

---

# ROOM DATABASE

## MENU 1 : DATA PENDUDUK

Buat Fragment baru:

```text
PendudukFragment
```

Menggunakan Room Database.

---

### Fitur

* Tambah Penduduk
* Edit Penduduk
* Hapus Penduduk
* Detail Penduduk
* Cari Penduduk

---

### Data Penduduk

* NIK
* Nama Lengkap
* Jenis Kelamin
* Tanggal Lahir
* Alamat
* RT/RW
* Pekerjaan

---

### Struktur Room

```text
PendudukEntity
PendudukDao
PendudukRepository
PendudukViewModel
NusaDataDatabase
```

---

### Tampilan

RecyclerView Card

Setiap card menampilkan:

* Nama
* NIK
* Alamat
* Tombol Detail
